# Little Shoots Plant Store Chatbot

Welcome to the Little Shoots Plant Store Chatbot repository! This project is a web-based chatbot designed to assist customers of the Little Shoots plant store. The chatbot provides information about plant sales, customer service, and general plant care by leveraging OpenAI's GPT-4.0-mini model.

## Features

- **Interactive Chat Interface**: A user-friendly chat interface with responsive design, enhanced with company branding and a seamless user experience.
- **PDF Content Extraction**: Extracts and indexes content from a PDF file (Questions.pdf) to provide accurate and relevant responses.
- **OpenAI Integration**: Utilizes OpenAI's GPT-4.0-mini model for generating responses based on user queries and indexed content.
- **Easy Deployment**: Built with Flask, making it simple to deploy and run on any server.

## Technologies Used

- **Python**: Backend logic and PDF content extraction.
- **Flask**: Web framework for serving the chatbot application.
- **OpenAI API**: For natural language processing and response generation.
- **PyPDF2**: For reading and extracting text from PDF files.
- **HTML/CSS/JavaScript**: Frontend interface design and functionality.
- **Font Awesome**: Icons for a more visually appealing interface.
